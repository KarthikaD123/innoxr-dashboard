# InnoXR Dashboard

A frontend prototype for the InnoXR HoloInteract Device Dashboard developed using React and Vite.

## Features

- Dashboard Overview
- Device Management
- Analytics Dashboard
- Settings Page
- Dark/Light Theme
- Responsive UI

## Tech Stack

- React.js
- Vite
- CSS3
- React Router
- React Icons

## Installation

```bash
npm install
npm run dev
```

## Project Structure

```
src/
  components/
  context/
  pages/
  assets/
```
